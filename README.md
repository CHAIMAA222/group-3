# group-3
🟢 Todo List Manager – Summary
🇬🇧 English

This is a simple command-line Todo List application built using Python.
It allows users to manage daily tasks efficiently by adding, viewing, updating, and deleting tasks.
All tasks are stored locally in a JSON file (todos.json), so the data remains saved even after closing the program.

Main features:
Add new tasks with priority (high, medium, low)
Display all tasks (completed and pending)
Mark tasks as completed
Delete tasks by ID
Save and load data automatically using JSON
Simple interactive menu in the terminal

This project is useful for beginners learning Python, file handling, and basic data structures.
